1. ค้นหาข้อมูลว่าแท็ก <!doctype> มีหน้าที่อะไร? (5 นาที)

Answer    ประกาศให้ Web Browser ทราบว่าเอกสารที่กำลังเปิดอยู่นั้นยึดหลักตามมาตรฐาน HTML 5



2.ถ้าอยากให้มี tooltip ใน tag นั้นต้องใช้ attribute ใด? (3 นาที)

Answer  ใช้ Attribute "title"


3. แท็ก <b> , <i> ต่างจาก <strong>, <em> หรือไม่ อย่างไร? (5 นาที)

Answer     Tag <b> กับ <i> ใช้เพื่อนำสายตาการมอง โดยไม่ได้เน้นหรือให้ความสำคัญกับข้อมูล
search engine จะมองเห็นเป็นข้อความธรรมดา ซึ่งต่างจาก Tag <strong> กับ <em> ที่ใช้ในการให้ความสำคัญของข้อมูล search engine จึงให้ความสำคัญมากกว่า
